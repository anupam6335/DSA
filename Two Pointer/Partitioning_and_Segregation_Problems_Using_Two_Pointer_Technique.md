# LeetCode Partitioning & Segregation Problems Using Two-Pointer Technique

## **Easy Problems**

1. **Move Zeroes** (283) - **Classic Problem**
   - Move all 0's to the end while maintaining relative order

2. **Sort Array By Parity** (905)
   - Move all even integers before odd ones

3. **Sort Array By Parity II** (922)
   - Rearrange so even indices have even numbers, odd indices have odd numbers

4. **Remove Element** (27)
   - Remove all occurrences of val in-place

5. **Remove Duplicates from Sorted Array** (26)

## **Medium Problems**

6. **Sort Colors** (75) - **Dutch National Flag Problem** ⭐
   - Sort array of 0's, 1's, and 2's in-place

7. **Partition List** (86)
   - Partition linked list around value x


8. **Partition Labels** (763)
   - Partition string so each letter appears in at most one part


9. **Wiggle Sort** (280) - Premium
   - Reorder in-place: nums[0] ≤ nums[1] ≥ nums[2] ≤ nums[3]...


10. **Wiggle Sort II** (324)
    - Reorder in-place: nums[0] < nums[1] > nums[2] < nums[3]...


11. **Partition Array Into Three Parts With Equal Sum** (1013)
    - Check if array can be partitioned into 3 parts with equal sum


12. **Separate Black and White Balls** (2938) - New
    - Move all '1's to right, '0's to left (minimum swaps)

13. **Rearrange Array Elements by Sign** (2149)
    - Alternate positive and negative numbers

14. **Sort Even and Odd Indices Independently** (2164)
    - Sort even indices ascending, odd indices descending

## **Hard Problems**

15. **First Missing Positive** (41)
    - Find smallest missing positive integer using in-place hashing

16. **Missing Number** (268)
    - Find missing number in range [0, n] without extra space

17. **Set Mismatch** (645)
    - Find duplicate and missing number using in-place marking

18. **Couples Holding Hands** (765)
    - Minimum swaps to arrange couples together

19. **Kth Largest Element in an Array** (215)
    - Find kth largest element using quickselect (partitioning)

20. **Smallest Range Covering Elements from K Lists** (632)
    - Partition across multiple sorted lists
